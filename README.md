# 密码学

密码学一词源於希臘語kryptós“隱藏的”，和gráphein“書寫”

现代密码学的形成有三个阶段。
1949年之前为古典密码学
1949年至1975年之间作为科学的一个分支
1976年之后由于解决了对称密码中通信双方共享密钥的问题，现代密码学有了很大的发展。 加密密钥和解密密钥相独立。
